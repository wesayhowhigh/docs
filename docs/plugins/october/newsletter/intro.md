# Newsletter Plugin

The JUMP Newsletter Plugin supports the following providers, **Campaign Monitor**, **Hub Spot** and **Mailchimp**.

## Installation

Type `composer require jump/oc-newsletter-plugin` into your project root terminal.

## Configuration

### Setup
#### Example of how to set different driver types.
```dotenv title=".env"
JUMP_NEWSLETTER_DRIVER=campaign_monitor
or
JUMP_NEWSLETTER_DRIVER=mailchimp
or 
JUMP_NEWSLETTER_DRIVER=hubspot
```
