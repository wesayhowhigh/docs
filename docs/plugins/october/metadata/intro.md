# Metadata Plugin

## Introduction

The JUMP Metadata Plugin allows you to add additional meta information and SEO features to the pages you build.

## Installation

Type `composer require jump/oc-metadata-plugin` into your project root terminal.


## Repository

[Github](https://github.com/wesayhowhigh/oc-metadata-plugin)
