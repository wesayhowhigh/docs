# Backend Plugin

## Introduction

The Backend Plugin provides a number of features to the October backend, including a number of Form Widgets, Versioning and Preview capabilities.

## Installation

Backend is a core JUMP plugin so you likely already have it installed.

If not, simply type:

`composer require jump/oc-backend-plugin`
