# Remove Trailing Slash

When enabled, ensures the trailing slash is removed from the site, for canoncial urls.

## Setup

`JUMP_REMOVE_TRAILING_SLASH=true|false`, defaults to `true`.

```dotenv title=".env"
JUMP_REMOVE_TRAILING_SLASH=true
```
