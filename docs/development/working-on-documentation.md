# Working on Documentation

### Installation

- Clone the [Repository](https://github.com/wesayhowhigh/docs)
- [Node.js](https://nodejs.org/en/download/) version 14 or above
- Recommend you install [Yarn](https://classic.yarnpkg.com/lang/en/docs/install/#mac-stable)

## Development

Run the development server:

```bash
cd docs
yarn
yarn run start
```
This will open up a development server on http://localhost:3000/

Any changes you make to the files inside /docs will trigger an automatic rebuild of the site and an automatic live update of the page in your browser!

## Creating a new Section

Simply create a new folder inside `/docs`
