# Code Intro
This section is to be used for coding best practices.

## PHP

All PHP Code should be written to [PSR12 standards](https://www.php-fig.org/psr/psr-12/).

## Javascript

All Javascript code should be written to [ECMAScript 6+](http://es6-features.org/) (aka ES6) standards.
